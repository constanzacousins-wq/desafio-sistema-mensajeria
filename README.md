# Sistema de Mensajería

Desafío realizado para el Módulo 2: CSS Avanzado del bootcamp Fullstack JavaScript en Desafío Latam.
El proyecto consiste en maquetar una interfaz de sistema de mensajería utilizando Flexbox, replicando un layout de contactos y una sección principal centrada.

---

## Tecnologías utilizadas

- HTML5
- CSS
- Flexbox
